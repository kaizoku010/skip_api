SK!P API Documentation

These are routes/endpoints that connect you to the SK!P server and Firebase/MongoDB for easy operations.

API Routes

Method Route Description Action
GET /api/events Fetch all events
Show Example

Request

GET /api/events

Response

{
  "events": [
    {
      "eventId": "1",
      "eventName": "Sample Event",
      "eventDate": "2024-08-08T00:00:00Z",
      "eventGraphicsURL": "https://example.com/event.jpg"
    }
  ]
}
GET /api/attendees Fetch all attendees
Show Example

Request

GET /api/attendees

Response

{
  "attendees": [
    {
      "uid": "1",
      "userName": "John Doe",
      "bio": "Enthusiastic attendee"
    }
  ]
}
GET /api/posts Fetch all posts
Show Example

Request

GET /api/posts

Response

{
  "posts": [
    {
      "postId": "1",
      "eventId": "1",
      "uid": "user1",
      "userName": "John Doe",
      "content": "This is a sample post",
      "imageURI": "https://example.com/image.jpg",
      "videoURI": "https://example.com/video.mp4"
    }
  ]
}
GET /api/user-posts/:userId Fetch posts created by a specific user
Show Example

Request

GET /api/user-posts/12345

Response

{
  "posts": [
    {
      "postId": "1",
      "eventId": "1",
      "uid": "12345",
      "userName": "Jane Smith",
      "content": "User's post content",
      "imageURI": "https://example.com/image.jpg",
      "videoURI": "https://example.com/video.mp4"
    }
  ]
}
GET /api/chat/:roomID Fetch chat messages from a specific chat room
Show Example

Request

GET /api/chat/12345

      "messageId": "1",

      "content": "Hello!",
      "timestamp": "2024-08-08T00:00:00Z"
    }
  ]
}

POST /api/signup Register a new user
Show Example

Request

POST /api/signup
{
  "userId": "newuser",
  "userName": "New User",
  "password": "password123"
}

Response

{
  "message": "User registered successfully",
  "userId": "newuser"
}
POST /api/add-attendee Add a new attendee
Show Example

Request

POST /api/add-attendee
{
  "userId": "user1",
  "userName": "Attendee",
  "bio": "Some bio"
}

Response

{
  "message": "Attendee added successfully",
  "userId": "user1"
}
POST /api/posts/:postId/like Like a specific post
Show Example

Request

POST /api/posts/12345/like

Response
